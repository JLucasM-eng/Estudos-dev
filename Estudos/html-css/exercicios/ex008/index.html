Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b.

é melhor usar a tag <strong>

Italico

Pra Italico é a mesma coisa. Essa tag i é de forma. É melhor usar a tag em

Texto marcado

Podemos usar a tag MARK para um texto marcado.

Pra mudar a cor do marcador ai ja estamos falando de forma, ai ja é css. Uma das formas de usar css em coisas pontuais, é alterando na prorpia tag usando o paramentro style. como nesse exemplo do codigo. É mais legal usar uma tag style de fora. pra mudar tudo. se for o caso. Mas se for pra alterar algo pontual como só essa marcação. ai pode ser desse jeito mesmo.


Envelopar trechos do texto em uma tag

Dica para envelopar um trecho de texto em uma tag: Selecione o texto, aperte ctrl + shift + p e digite abb. Clique em Wrap whith abbreviation e digite qual a tag que voce quer usar.

Textos pequenos e grandes.

Para textos grandes usa-se a tag big. mas ela ja está fora de uso. Já pra texto pequeno pode usar a tag small que ainda ta usando.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerao.

Texto inserido

Podemos marcar um texto como inserido para dizer que ele foi inserido depois.
