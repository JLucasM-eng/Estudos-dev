Listas Ordenadas

Usa-se a tag ol. Cada item vai ser uma tag li. Voce pode mudar o tipo ou de onde começar mudando os parametros dentro da tag ol. tipo <ol type = "i">

  1. Acordar
  2. Tomar cafe
  3. Escovar os dentes

Listas não Ordenadas

Listas que eu n preciso numerar. Use a tag ul. e cada item sera um li. tambem posso mudar o tipo nos parametros.

  • Pão
  • Queijo
  • Frango

Lista de definição

Aqui voce usa pra listar o termo es definições desse termo. pra iniciar a lista usa a tag dl. para listar cada termo usa dt. Para colocar a descrição do termo usa a dd.

Descrição
