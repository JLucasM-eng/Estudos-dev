Codigo-fonte

Pra trocar a fonte para uma fonte mais legal como essa do exemplo a seguir, use a tag code. O comando document.getElementbyId é um comando de JS.
Essa tag não respeita a identação, por exemplo, no codigo desse doc o trecho de python a seguir esta todo bonitinho, mas aqui ta tudo junto: num = int(input("Digite um numero:")) if num % 2 == 0: printf('O numero {num} é par ') else printf('O numero {num} é impar.')
Precisamos entao usar tudo isso, ate a propria tag code dentro de uma outra tag pre. Veja no codigo. 

    
      num = int(input("Digite um numero:"))
      if num % 2 == 0:
        printf('O numero {num} é par ')
      else
        printf('O numero {num} é impar.')

Essa tag pre respeita tudo q ta dentro dela, todos os espaços e tabs.
